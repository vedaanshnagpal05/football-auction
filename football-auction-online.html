⚽ Online Auction — one-time setup

This page needs a free Firebase project to sync bids between devices. Takes ~5 minutes, no credit card.
1. Go to console.firebase.google.com → Add project (any name, Analytics off is fine).
2. In the left menu: Build → Realtime Database → Create database → choose a location → start in test mode.
   (Test mode is open for 30 days — fine for casual games. To keep it open longer, set rules to {"rules":{"rooms":{".read":true,".write":true}}}.)
3. Click the ⚙️ gear → Project settings → under "Your apps" click the </> web icon → register the app (no hosting needed).
4. Copy the firebaseConfig = {...} object it shows you.
5. Open this HTML file in a text editor, find const FIREBASE_CONFIG near the top of the script, and paste your values over the placeholders. Save.
6. To play with friends on other devices, put the file online: drag it into app.netlify.com/drop (free, instant URL) or use GitHub Pages. Share that URL — everyone opens it, one person hosts a room and the rest join with the 4-letter code.
Tip: the offline version of this game (pass-and-play + AI + manual auctioneer) needs none of this.
